# Leaner.js

Leaner.js is a thin, reactive wrapper for DOM. It lets you create modern client-side JavaScript applications without compilers or build tools.


## Getting Started

Documentation: https://leanerjs.org/


## Development

To build the package:

```bash
npm run build
```

To run tests:

```bash
npm test
```
